About FTS:

FILE TRACKING SYSTEM (FTS) is a web application that manages all the files movement from one desk to another one and helps them in managing the flow of files effectively and efficiently. All the files such as: reports, decisions, requests,reminders, and others can be processed and tracked by the system at any time.

Follow bellow step to Create New Employee your department:-

  1. Login with Head of Department(HOD) user

  2. Go to "User Management" menu option

  3. Click on "Add User" button at upper left corner on page

  4. Select Circle,Division,Subdivision,Department,Designation,Role etc.

  5. Enter Emp Id, Name, ISMS login Id, email, mobile no and select active as "Yes"

  6. Click on "Save Changes" button

Follow bellow step when Employee transfer or retire:-

  1. Clear in process files in employee login by transfer files to other user

  2. Login with HOD user

  3. Go to "User Management" menu and search transferd employee by emp number

  4. Click on "Edit" button then select "Active" as "No" and provide Deactive reason

  5. Click on "Save Changes" button
